Instructions

Suppose you want to perform 5 actions on the given web page using a voice interface (such as Siri and Alexa). For each action:

  • Click Highlight Element and then click on the element (textbox, button, etc.) on the web page that the action acts on.
  • Write down a command you would say to the voice interface.

Examples

  • highlight the Search button → "click submit"
  • highlight the textbox under the "To:" label → "enter LAX into the destination field"
  • highlight the article title → "click on the first article" or "go to the article by trever"
  • highlight the textbox near the magnifying glass icon → "in the search box, type jacket"

Guidelines

  • When writing the commands, imagine that you need to do the task without clearly seeing the texts on the web page.
  • As such, try to avoid the exact words that appear on the web page (unless they are the only sensible words, such as "password"). We give $0.02 bonus for each command that does not use the exact words from the element.
  • However, be specific enough that the voice interface can guess the single right target.
  • Try to vary the sentence structures.
  • Rejection: We manually review the submissions and will reject works that are repetitive (e.g., automatically generated from a template), nonsensical (e.g., random words), or incomplete.
