Instructions

  1. On the given web page, come up with a task that uses the web page (e.g., search for flights, register an account, select a product).
  2. Select 5 elements on the page that are relevant to the task.
  3. For each element, write down 3 different ways to refer to it in English.
    • Tip: Imagine that you need to do the task without seeing the web page. You have to tell your friend (or your personal assistant) to do the task for you. Try to name the element so that your friend can choose the right element on the web page.
    • Example 1: A "Login" button = "login button", "submit", and "sign in button"
    • Example 2: A "Password" input box = "password box", "password field", and "pass"
    • Example 3: A button with the magnifying glass icon = "search", "go", and "submit button" ("magnifying glass" and "magnifying glass icon" are also fine)
    • Please try to avoid the exact words that appear on the web page, except when it is absolutely necessary.
