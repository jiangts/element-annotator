Instructions

Suppose you want to perform 5 actions on the given web page using a voice interface (such as Siri and Alexa). For each action:

  • Write down a command you would say to the voice interface.
  • Click Highlight Element and then click on the relevant element (textbox, button, etc.) on the web page to highlight it.

Examples

  • "enter LAX into the destination field" → highlight the textbox under the "To:" label
  • "click submit" → highlight the Search button
  • "click on the first article" → highlight the article title
  • "in the search box, type jacket" → highlight the textbox near the magnifying glass icon

Guidelines

  • When writing the commands, imagine that you need to do the task without clearly seeing the texts on the web page.
  • As such, try to avoid the exact words that appear on the web page (unless they are the only sensible words, such as "password").
  • But also be specific enough that the voice interface can guess the right target.
  • Try to vary the sentence structures.
